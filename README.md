## Project Description and Hosted Links

## Project 1: Local Community Event Portal (HTML5, CSS3 & JavaScript)

**Description:**
Developed a responsive Local Community Event Portal using HTML5, CSS3, and JavaScript. The application enables users to browse community events, register for events, view event galleries, watch promotional videos, store preferences using local storage, and search for nearby events. The project demonstrates form validation, DOM manipulation, event handling, asynchronous JavaScript operations, and responsive web design.

**Hosted Project Link:**
[https://projecthtmlcssjsfirst.netlify.app/](https://projecthtmlcssjsfirst.netlify.app/)

### Project 2: Bootstrap 5 Responsive Website

**Description:**
Developed a responsive website using Bootstrap 5 to showcase modern front-end development techniques. The project includes responsive grid layouts, navigation bars, forms, cards, buttons, typography utilities, Bootstrap Icons, modals, accordions, and other UI components. The website is designed with a mobile-first approach and adapts seamlessly across different screen sizes.
